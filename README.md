# tiktok-src
Tiktok downloader & auto upload to Reels Facebook (with Puppeteer).

## Install
1. Akses Facebook Web (Pastikan Anda sudah login terlebih dahulu).
2. Ekspor cookies Anda menggunakan ekstensi [Export Cookies JSON file for Puppeteer](https://chrome.google.com/webstore/detail/%E3%82%AF%E3%83%83%E3%82%AD%E3%83%BCjson%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB%E5%87%BA%E5%8A%9B-for-puppet/nmckokihipjgplolmcmjakknndddifde).
3. Clone repositori ini: [https://github.com/wahdalo/tiktok-src](https://github.com/wahdalo/tiktok-src).
4. Buka direktori dari repositori ini.
5. Ganti file `cookies.json` dengan file yang diekspor dari ekstensi.
6. Instal modul dengan perintah `npm install`.

**Catatan:** Masukkan link Tiktok seperti di sini.

## Fitur Bot Telegram (Opsional)
1. Untuk menghubungkan ke Bot Telegram chat, pertama-tama buat bot di [BotFather](https://t.me/BotFather).
2. Salin token dari BotFather ke dalam file `config.json`.
3. Jalankan bot di terminal dengan perintah `node telebot.js`.
4. Tempelkan URL video Tiktok Anda ke dalam bot Telegram yang telah Anda buat.
